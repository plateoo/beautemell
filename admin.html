Comment obtenir votre token

1

Rendez-vous sur developers.facebook.com → Mon appli → Créer une appli → Choisir "Entreprise"

2

Dans l'appli, ajoutez le produit Instagram Basic Display. Allez dans "Génération de token" et connectez votre compte Instagram.

3

Copiez le token généré (valide 60 jours) et collez-le ci-dessus. Pour un token permanent, utilisez l'API Graph avec une page Facebook liée.

🌸 Entrez votre token Instagram et cliquez "Charger" pour afficher vos publications.
