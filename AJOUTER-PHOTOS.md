# Comment ajouter les vraies photos de Mell

Facebook bloque le telechargement automatique. Voici comment faire manuellement :

## Etapes

1. Va sur Facebook : https://www.facebook.com/profile.php?id=61572533833331
2. Clique sur l'onglet **Photos**
3. Clique sur chaque photo d'ongles que tu veux ajouter
4. **Clic droit** sur la photo > **Enregistrer l'image sous**
5. Sauvegarde dans : `C:\Users\Tunc jordan\Desktop\beautemell\photos\`
6. Nomme-les : `photo-1.jpg`, `photo-2.jpg`, `photo-3.jpg`, etc. (jusqu'a `photo-8.jpg`)
7. Double-clique sur **add-photos.bat** pour publier sur le site

## Notes

- Les photos doivent etre en `.jpg` ou `.png`
- Minimum 8 photos pour remplir le portfolio
- `photo-profil.jpg` = la photo de Mell dans la section "A propos"
- Le site utilise les images dans `photos/photo-1.jpg` a `photos/photo-8.jpg`
